Checkered Sweater

I was inspired to make a checkered pattern after seeing a similar design Instagram. I love that it is all one color and a very textural pattern. I used Lion Brand's Pound of Love yarn and size 7 needles.

This was a super simple sweater to make since it is a such repetitive pattern. I just alternated between five knit stitches and five purl stitches for five rows. This gives a checkered appearance even though it is only one color.

When I was making the sweater, I didn't need to block the sweater to flatten it because the fabric was already pretty flat. There was no curling at the edges that you would get when using a stockinette stitch. The reason stockinette curls is because the stitches don't lay evenly on the fabric. However, alternating between knit and purl stitches evens everything out so the fabric stays flat.

This is a portfolio project for the Fall 2022 semester of Web Design.
